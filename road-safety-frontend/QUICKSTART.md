# 🚀 快速啟動指南

## 步驟 1: 啟動後端 API

在專案根目錄執行：

```bash
python backend.py
```

確認後端運行在 `http://localhost:5001`

## 步驟 2: 啟動前端應用

在 `road-safety-frontend` 目錄執行：

```bash
cd road-safety-frontend
npm start
```

## 步驟 3: 開啟瀏覽器

自動開啟 `http://localhost:3000`，或手動在瀏覽器輸入該網址。

## 預期結果

你應該會看到：

1. ✅ 頂部顯示「道路安全地圖」標題
2. ✅ 區域安全評估卡片（顯示總分、道路數、監視器數、捷運站數）
3. ✅ 圖例說明（綠/黃/紅色代表的意義）
4. ✅ 互動式地圖，顯示台北市政府附近的道路
5. ✅ 道路以不同顏色標示（根據安全分數）
6. ✅ 點擊道路可查看詳細資訊

## 測試功能

### 1. 查看道路詳情
- 點擊地圖上的任一道路線條
- 會彈出視窗顯示：
  - 道路名稱
  - 道路類型
  - 安全分數
  - 監視器數量
  - 捷運站數量

### 2. 地圖操作
- **縮放**: 使用滑鼠滾輪或右上角的 +/- 按鈕
- **拖曳**: 按住滑鼠左鍵拖動地圖
- **重置**: 重新整理頁面回到初始位置

### 3. 查看不同區域
修改 `src/App.js` 中的中心點座標：

```javascript
// 台北車站
const [center] = useState({ lat: 25.0478, lng: 121.5170 });

// 信義區
const [center] = useState({ lat: 25.0330, lng: 121.5654 });

// 士林夜市
const [center] = useState({ lat: 25.0876, lng: 121.5240 });
```

## 常見問題

### Q: 地圖顯示空白？
A: 檢查瀏覽器控制台是否有錯誤，確認後端 API 正常運行。

### Q: 沒有顯示道路？
A: 
1. 確認後端 API 有返回資料
2. 檢查 Network 標籤查看 API 回應
3. 嘗試重新整理頁面

### Q: 顏色都是紅色？
A: 這表示該區域監視器和捷運站較少，可以：
1. 換到市中心區域測試
2. 調整 `safety_radius_m` 參數增加檢測範圍

### Q: API 連線錯誤？
A: 
1. 確認後端運行在 port 5001
2. 檢查防火牆設定
3. 確認沒有其他程式佔用 port

## 進階設定

### 修改搜尋範圍

在 `src/App.js` 中：

```javascript
const searchRadius = 1000;  // 增加到 1000m
const safetyRadius = 300;   // 增加到 300m
```

### 修改安全分數門檻

```javascript
const getColor = (score) => {
  if (score >= 30) return '#22c55e'; // 提高安全門檻
  if (score >= 15) return '#eab308'; // 提高注意門檻
  return '#ef4444';
};
```
